# Physical Plausibility Prediction for synthetically generated videos
Physical Plausibility Prediction for evaluating whether the physical interactions and behaviors observed in synthetically generated videos from world models are consistent with real-world physics laws and constraints.

### Example usage:  Run inference on videophy2 dataset using offline mode.

    uv run examples/video_critic/inference_videophy2.py --model nvidia/Cosmos-Reason2-2B
